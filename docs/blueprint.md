# **App Name**: YashDev Freelance Hub

## Core Features:

- Animated 3D Landing Page: Interactive 3D animation on the landing page to showcase services.
- Service Showcase: Detailed display of freelancing services including Android app development, desktop app development, AI/ML model development, LLM development, and web development.
- About Us with AI content Tool: An 'About Us' section that can be expanded or shortened using an AI content tool.  This content will include project details, platform specializations and competitive pricing.
- Contact Form with Email.js Integration: A contact form that uses Email.js to send user input directly to YashDev. Fields will include name, email, project category, and contact number.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#EEEEEE) to ensure content clarity and modern interface aesthetic.
- Accent color: Vibrant blue (#29B6F6) to draw attention to interactive elements.
- Font: 'Inter' (sans-serif) for both headlines and body text. Ensures readability and a clean, modern look.
- Simple, line-based icons to represent services offered. Icons should integrate smoothly within the 3D animated environment.
- Clean and modern layout with ample white space to avoid clutter and emphasize content. Use grid system for consistent alignment.
- Smooth transitions and subtle animations to enhance user experience without being distracting. The 3D animation should be interactive and responsive to user input.
